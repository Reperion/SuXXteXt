# Social Preview Image Idea: "SuxxText"

**Goal:** Create a 1280x640 pixel image for the GitHub "Social Preview" card (Settings -> General -> Social Preview).

## Image Prompt Concept

**Visual Style:** Cyberpunk / Terminal / Matrix Aesthetic (Green on Black)
**Text Overlay:** "SuxxText: YouTube -> Text"

**AI Image Generation Prompt:**
> A futuristic, glowing green digital waveform transforming into lines of code and text script. Dark background with matrix-style digital rain. In the center, a bold, glitch-effect title reading "SuxxText". High contrast, tech-noir style, 8k resolution, cinematic lighting.

## Implementation Tips
1. Use Midjourney, DALL-E, or Ideogram with the prompt above.
2. Overlay the text "Offline YouTube Transcription & Analytics" in a clean sans-serif font (like Inter or Roboto) if the AI text is messy.
3. Upload to GitHub Repo Settings!
